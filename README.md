# z180ISPAdapter
Adapter board between s100computers z180 SBC board and z80-Retro! in-system programmer

Version 1 has incorrect pinout for connecting to the programmer, but can be made to work with dupont wires.  Also missing some pullup resistors, HOWEVER, the adapter seems to work fine WITHOUT the resistors installed.

Version 2 has fixed the pinout and additional pullup resistor pads, but it has not been tested.
